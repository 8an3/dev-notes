### Editor / Viewer

Global snippet management system built on Monaco editor for creating, editing, and viewing snippets


**Access:** Extensions title pane or web GUI

**Features:**
- Full Monaco editor integration
- Create and edit snippets with syntax highlighting
- View all global snippets in one place
- Fuzzy search capabilities
- Quick insertion workflow
- Automatic workspace synchronization

### Context Snippets

Access and insert snippets directly from the editor context menu with integrated search.

**Access:** Right-click → DevStack → Context Snippets

**Features:**
- Insert snippets without leaving your editor
- Built-in search functionality
- Language-specific snippet filtering
- Support for variables and placeholders
- Quick access to frequently used code blocks
